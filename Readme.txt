C'est un projet que j'ai commencé avec une cliente dans le cadre de ma formation pour le titre de développeur web web mobile, ce projet est en cours de développement.
La cliente souhaite un site web client ainsi qu'une partie back-office pour gérer la partie client. C'est un site de réservations d'ateliers de couture dispensé par la cliente, 
avec une partie blog où elle partage ses autres passions. Pour réserver une place à une atelier, il faut que le visiteur se crée un compte. La partie client est également composée d'une page 
de contact avec un formulaire pour des demandes particulières, une page blog, une page où l'on peut voir tous les ateliers disponibles, une page avec les photos des créations lors des ateliers,
ainsi qu'une page de de connexion/ creation espace client, et un espace client.

La partie back-office est sécurisée par une adresse email et un mot de passe hashé en base de donnée. Pour l'instant seuls les modules clients/ utilisateurs(admin)/ créations (CRUD) 
sont opérationnels.
